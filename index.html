Yet Another S4 Blog (YASB)

Introduction

The aim of this blog is to aid those who wish to learn the basic mathematical background of and to learn how to implement the Structured State Space for Sequence Modeling (S4) architecture. This will be done in two sections: (1) mathematical background and (2) code implementation in Python and Pytorch.

Mathematical Background

The S4 architecture takes inspiration from the linear time invariant (LTI) dynamical system, where a hidden state is updated using data inputs, which is then used to evolve the system with linear operators.

\[ \sum_{n=1}^{\infty} \frac{1}{n^2} = \frac{\pi^2}{6} \]
